# RGB is 3D

Visualize the colors of an image as a 3D RGB point cloud.

[Try it.](https://yig.github.io/image-rgb-in-3D)
(Requires WebGL.)

The image to load can be passed as an 'image' GET parameter or uploaded directly from your computer.
